Reading a Soft Keyboard via Scanning

Pressing one of the buttons in the keypad below should show the corresponding key in the area below:

The animation shows the scanning of the rows. When a key is pressed, a one-hot encoding of the row and column index will be passed to the row and col fields of the ScanKeys struct on the next scan. Your code should decode these indexes to return the corresponding char in the out field.
